  • Persists user preference in localStorage
  • Respects system preference (prefers-color-scheme)
  • Toggle button in top-right corner (or header)
  • No flash or jarring color changes
  • Optimized shadows for both themes
Dark Mode Toggle Implementation
// Toggle function
function toggleTheme() {
    const isDark = document.body.classList.toggle('dark-mode');
    localStorage.setItem('theme', isDark ? 'dark' : 'light');
    updateShadows(); // Adjust shadows for theme
}

// Initialize from saved preference
const savedTheme = localStorage.getItem('theme');
if (savedTheme === 'dark' ||
    (window.matchMedia('(prefers-color-scheme: dark)').matches && !savedTheme)) {
    document.body.classList.add('dark-mode');
}
